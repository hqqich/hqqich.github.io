概述

1Remote 是一款面向 Windows 的开源远程会话管理工具，支持 RDP、SSH、VNC、Telnet、(S)FTP、RemoteApp 等多种协议。软件设计为绿色便携（无需安装，解压即可运行），界面基于选项卡的会话管理，方便在多个远程主机之间切换和组织连接。

地址

github.com/1Remote/1Remote

主要功能（详述）

  • 开源且免费：源代码在 GitHub 上公开，可自行查看、编译或参与贡献。
  • 多协议支持：RDP（远程桌面）、SSH（命令行/隧道）、VNC、Telnet、(S)FTP、RemoteApp 等，满足常见远程访问需求。
  • 多屏与高分辨率支持：对多显示器、高 DPI 场景的 RDP 支持更好，能够保持显示效果与缩放正确性。
  • 快捷键与一键连接：内置快捷键（例如：Alt + M）可以快速打开常用的远程连接，提高效率。
  • 可在单个服务器配置中保存多个地址与账号：同一服务器下可维护多个登录凭据或不同端口的目标，便于运维场景管理。
  • 选项卡式会话界面：每个远程会话在选项卡中打开，方便在多个会话间切换而不占用额外窗口。
  • 多语言与主题：支持多语言界面，可自定义颜色主题以适配个人习惯或团队规范。
  • 绿色部署：无需安装，直接解压即可运行，便于携带与部署到受限环境。

优势

  • 轻量与便携：相比重量级商用工具，1Remote 更轻便，启动快，适合作为工具箱中的便携客户端。
  • 集中管理多种协议：不用来回切换不同客户端，统一管理远程会话能提升工作效率。
  • 可审阅与自托管：开源的性质允许企业或个人审计代码、按需定制或在内部网中部署。

安装与快速上手

  1. 下载
    • 到项目 GitHub 页面下载最新的 release 压缩包（通常为 zip）。
  2. 解压
    • 将压缩包解压到任意目录（例如 D:\tools\1Remote）。
  3. 运行
    • 双击主程序（通常为 1Remote.exe 或类似文件名），软件会以绿色便携方式启动。
  4. 添加连接
    • 在左侧或顶部的“新建连接”/“Add”处填写主机地址、端口、协议类型与用户名/密码（或使用密钥文件进行 SSH 认证）。
  5. 保存与组织
    • 可以把连接按分组、标签或目录保存，方便日后快速查找。
  6. 快速连接
    • 选中一个连接后双击或使用定义的快捷键（如 Alt + M）即可快速打开。

配置管理提示

  • 对于 SSH，建议使用密钥对认证并在配置中引用私钥文件（如果支持），提高安全性。
  • 为常用连接启用“记住密码/凭据”要慎重，敏感环境下建议使用系统凭据管理或不保存密码。
  • 如果需要在公司网络内部部署，确认防火墙与端口策略允许所需协议通过。

使用技巧

  • 标签/分组：把同一项目或同一环境的机器放到同一组，切换更快。
  • 模板/批量导入：如果 1Remote 支持导入 CSV/JSON 配置，可通过导入快速添加大量服务器。
  • 会话断线重连：在网络不稳定时，优先查看是否有自动重连或会话保存选项以减少重复认证步骤。
  • 快捷键自定义：熟悉并自定义快捷键（如锁定、截图、断开连接）可以极大提升操作效率。

常见问题（FAQ）

Q: 1Remote 是否支持 macOS 或 Linux？

A: 目前 1Remote 主要面向 Windows 平台。如需跨平台使用，可查看项目仓库是否有计划或社区分支支持其它平台，或使用 Wine/Crossover 等兼容层尝试运行（兼容性不保证）。

Q: 是否可以共享配置或导出连接列表？

A: 大多数远程会话管理工具提供导出/导入配置的功能，便于团队共享。请查看仓库文档或软件中的“导入/导出”选项。

Q: 如何提高 RDP 在高 DPI 多屏下的显示效果？

A: 尽量让客户端与远程系统的缩放设置匹配，或者在 RDP 配置中禁用显示缩放选项以避免模糊。具体设置参见软件的显示/缩放选项。
